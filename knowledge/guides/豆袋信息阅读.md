# 豆袋信息阅读指南

豆袋上的信息决定了系统能不能理解一支豆子。看懂标签后，你也更容易判断这支豆该怎么冲、该期待什么风味。

## 先看这些字段

| 字段 | 说明 |
| --- | --- |
| 烘焙商 | 谁烘焙并销售这支熟豆 |
| 产品名 | 烘焙商给这支豆或拼配起的名称 |
| 产地 | 国家、地区、子产区，例如埃塞俄比亚古吉、巴拿马波奎特 |
| 来源主体 | 庄园、生产者、处理站、合作社等 |
| 生豆商 | 负责采购、进口或流通的机构，不等同于庄园 |
| 生豆批次或项目 | 生豆商用于组织豆源的批次、产品线或项目名，例如 CM Selections、Crown Jewels |
| 品种 | 瑰夏、波旁、SL28、卡杜拉等 |
| 处理法 | 水洗、日晒、蜜处理、厌氧、CM 等 |
| 烘焙日期 | 决定养豆、保存和起手参数 |
| 官方风味 | 烘焙商给出的香气和口味描述 |
| 官方建议参数 | 如果有，应优先记录为这支豆的参考配方 |

## 哪些信息最容易混淆

“庄园”和“生豆商”不是一回事。庄园、生产者、处理站、合作社描述咖啡从哪里生产和处理；生豆商、进口商、寻豆机构描述咖啡如何流通到烘焙商手里。

“生豆商”和“生豆批次”也不是一回事。前者是机构，后者是机构推出或管理的具体豆源项目、批次或产品线。它能帮助判断这支豆的来源体系，但不能替代产地、庄园、品种和处理法。

“品种”和“处理法”也不是一回事。品种是咖啡树材料，处理法是采收后的处理方式。瑰夏水洗、瑰夏日晒、瑰夏厌氧可能是完全不同的杯中表现。

## 对冲煮有什么帮助

如果标签显示浅烘、水洗、高海拔，起手可以偏清晰、偏高温；如果是日晒、厌氧或发酵感强的豆，起手可稍降温、减少扰动，先控制发酵感和涩感。真正的建议仍要结合器具、研磨和上一杯结果。
