# 器具与冲煮方法选择

器具会改变萃取方式、粉床形状、过滤效果和可控变量。系统给冲煮建议前，必须先知道你用的器具，否则只能给很泛的起手参数。

## 常见器具怎么选

| 器具 | 适合什么目标 | 建议时必须确认 |
| --- | --- | --- |
| V60 | 清晰度、花香、酸质和高可调性 | 滤纸、粉量、注水方式、是否容易绕流 |
| 平底滤杯 | 更稳定、甜感更集中、容错率较高 | 滤杯型号、滤纸、底部流速是否偏慢 |
| Pulsar / no-bypass | 减少绕流，兼顾浸泡和渗滤 | 阀门开合、水位、浸泡时间 |
| 爱乐压 | 便携、快速、宽容度高 | 正压或反压、浸泡时间、滤纸或金属滤 |
| 法压 | 厚重口感、低门槛、适合多人分享 | 粗研磨、浸泡时间、是否搅拌和过滤沉淀 |
| Chemex | 干净、轻盈、适合较大杯量 | 厚滤纸、排气口、总水量和流速 |
| 摩卡壶 | 浓郁、适合奶咖或意式替代 | 粉量是否压实、火力、出液后是否及时离火 |
| 意式机 | 浓缩、奶咖、高强度风味 | 粉量、粉饼准备、压力、时间和流速 |
| 冷萃 | 低酸、顺滑、适合提前制作 | 粉水比、温度、浸泡时间和过滤方式 |
| 冰手冲 | 保留热萃香气，又得到清爽冰咖啡 | 热水和冰块比例、研磨是否略细 |
| 挂耳 | 方便、无需磨豆机，适合办公室和旅行 | 包装新鲜度、水温、注水量和浸泡时间 |

## 起手判断

想要花香和清晰度，优先 V60、Chemex 或干净的平底滤杯；想要甜感和稳定性，平底滤杯、Pulsar、爱乐压更友好；想要厚重口感，法压和摩卡壶更直接；想做奶咖，意式机和摩卡壶更合适。

同一支豆换器具后，参数不能照搬。V60 的研磨和注水逻辑不等于 Chemex；爱乐压和法压属于浸泡逻辑；摩卡壶不是意式浓缩，但也不是普通手冲。

如果想快速理解某一种方式，可以直接看对应页面：[冰手冲咖啡指南](../brewing/冰手冲咖啡指南.md)、[冷萃咖啡指南](../brewing/冷萃咖啡指南.md)、[摩卡壶冲煮指南](../brewing/摩卡壶冲煮指南.md)、[法压壶冲煮指南](../brewing/法压壶冲煮指南.md)、[挂耳咖啡冲泡指南](../brewing/挂耳咖啡冲泡指南.md)、[意式饮品基础](../brewing/意式饮品基础.md)。

## 起手参数参考

| 方法 | 粉水比 / 浓度 | 研磨 | 时间 |
| --- | --- | --- | --- |
| V60 | 1:15 到 1:17 | 中细 | 2:30 到 3:30 |
| 平底滤杯 | 1:15 到 1:17 | 中细到中等 | 2:45 到 3:45 |
| Pulsar | 1:16 到 1:17 | 中等或略粗 | 3:30 到 4:30 |
| 爱乐压 | 1:12 到 1:16 | 中细到中等 | 1:30 到 3:00 |
| 法压 | 1:14 到 1:16 | 粗 | 4:00 以上 |
| Chemex | 1:15 到 1:17 | 中等到中粗 | 4:00 到 6:00 |
| 摩卡壶 | 装满粉碗，不压实 | 接近意式但略粗 | 出液完成即离火 |
| 意式浓缩 | 1:2 左右出液比 | 细 | 25 到 35 秒 |
| 冷萃 | 1:8 到 1:12 | 中粗到粗 | 8 到 16 小时 |
| 冰手冲 | 总液体 1:15 到 1:16 | 比热手冲略细 | 2:00 到 2:40 |
| 挂耳 | 1:14 到 1:16 | 预磨粉 | 1:30 到 2:30 |

这些范围只用于第一杯。浅烘水洗可略升温或调细；日晒、厌氧和深烘可先降低水温、减少扰动，避免发酵感和涩感过冲。

## 给系统提问时怎么说

最好直接说明器具、滤纸或滤网、粉量、水量、研磨、温度、时间和杯中结果。比如：“V60，15g 粉，240g 水，93°C，C40 20 格，2 分 30 秒，喝起来酸薄”，比“这支豆怎么冲”更容易得到有效建议。
