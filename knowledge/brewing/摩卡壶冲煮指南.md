# 摩卡壶冲煮指南

> 摩卡壶用蒸汽压力推动热水穿过咖啡粉，做出的咖啡浓郁、适合加奶，但它不是意式浓缩。

## 核心理解

摩卡壶由下壶、水管、粉碗、上壶组成。加热后，下壶水温升高并产生压力，热水被推上水管，通过粉碗萃取咖啡，再进入上壶。

摩卡壶的压力低于意式机，无法稳定达到真正意式浓缩的萃取条件。但它能做出比手冲更浓郁、比法压更干净的咖啡，适合做黑咖啡、加奶咖啡和家用浓缩替代。

## 起手参数

| 参数 | 建议 |
| --- | --- |
| 水量 | 加到安全阀下方，不要盖住安全阀 |
| 粉量 | 装满粉碗，刮平即可 |
| 研磨 | 比意式粗，比手冲细 |
| 压粉 | 不压实，只轻轻铺平 |
| 火力 | 中小火，避免猛烈加热 |
| 结束点 | 出液变浅、出现明显气泡声时离火 |

粉碗大小由摩卡壶决定，不建议为了“少喝一点”只装半碗粉。半碗粉容易形成不稳定通道，导致味道薄、苦或不均匀。

## 基础流程

1. 下壶加水到安全阀下方。想缩短加热时间，可以加入热水，但操作时要防烫。
2. 粉碗装满咖啡粉，轻轻刮平，不要压粉。
3. 擦干粉碗边缘，拧紧上下壶。
4. 用中小火加热，让咖啡稳定流入上壶。
5. 当出液颜色明显变浅，或开始出现喷溅和气泡声时，立刻离火。
6. 可用湿毛巾或冷水给下壶降温，停止继续萃取。
7. 轻轻搅匀上壶中的咖啡再倒出。

## 选豆思路

摩卡壶适合中烘到中深烘，也适合巧克力、坚果、焦糖、香料、熟果调性的豆子。浅烘豆也可以使用，但更容易出现酸尖、萃取不足或口感薄的问题，需要更细研磨、更高水温和更稳定火力。

如果要做奶咖，优先选择甜感和厚度明确的中深烘或意式拼配。如果想做黑咖啡，可以尝试中烘单品，但不要期待它表现得像 V60 一样轻盈清晰。

## 常见问题

| 问题 | 可能原因 | 下一次调整 |
| --- | --- | --- |
| 苦、焦、烟感 | 火太大，离火太晚 | 降低火力，出液变浅就离火 |
| 酸、薄 | 研磨太粗或萃取不足 | 略调细研磨，保持稳定加热 |
| 喷溅严重 | 火力太猛，或粉层阻力异常 | 降火，检查是否压粉或堵塞 |
| 没有油脂 | 摩卡壶本来不等于意式机 | 不以 crema 判断成功与否 |
| 咖啡渣进入上壶 | 研磨太细，或滤片密封不佳 | 略调粗，检查胶圈和滤片 |

## 不建议做的事

- 不要压粉。摩卡壶没有意式机的泵压，压实后容易堵塞和喷溅。
- 不要盖住安全阀。安全阀是泄压保护。
- 不要大火猛煮到底。后段高温和蒸汽会带来焦苦。
- 不要用“有没有油脂”判断咖啡是否好喝。
