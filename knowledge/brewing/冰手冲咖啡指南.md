# 冰手冲咖啡指南

> 冰手冲是“热水萃取 + 冰块快速降温”。它保留热萃的香气和层次，又能得到清爽低温的口感。

## 核心逻辑

冰手冲不是把一杯热咖啡放凉，也不是冷萃。它的关键是：把总水量的一部分换成冰块放在下壶，冲煮时只用剩余热水萃取。热咖啡滴入冰块后迅速降温，同时被融化的冰块稀释到目标浓度。

由于热水用量变少，咖啡粉和热水的接触量也减少。为了避免萃取不足，冰手冲通常需要比同一支豆的热手冲 **略细研磨、略高水温、节奏更集中**。

## 起手参数

| 参数 | 建议值 |
| --- | --- |
| 粉量 | 15g |
| 总水量 | 225 到 240g |
| 热水 | 135 到 150g |
| 冰块 | 75 到 105g |
| 水温 | 92 到 96°C |
| 研磨 | 比同豆热手冲略细 |
| 总时间 | 2:00 到 2:40 |

示例：**15g 粉 + 150g 热水 + 90g 冰块**。总液体约 240g，接近 1:16。浅烘水洗豆可用 94 到 96°C；日晒、厌氧或深烘可从 90 到 93°C 起步。

## 基础流程

1. 在分享壶中加入冰块。
2. 滤纸预润后倒掉润湿水，避免稀释比例失控。
3. 加入咖啡粉并整平粉床。
4. 用约 2 到 3 倍粉重的热水闷蒸 25 到 35 秒。
5. 分 2 到 3 段注完剩余热水。
6. 咖啡液滴入冰块后轻摇分享壶，让冰块快速融化均匀。
7. 倒入杯中，必要时补少量冰块调整温度。

## 选豆思路

冰手冲更适合香气清晰、酸质明亮、甜感明确的豆子。水洗埃塞、肯尼亚、巴拿马、哥伦比亚高甜感批次，都容易做出干净清爽的表现。日晒和厌氧豆也可以做冰手冲，但要注意发酵感在低温下可能更突出，建议降低水温或减少后段扰动。

深烘和中深烘也能做冰手冲，但目标通常不是花香和酸质，而是巧克力、坚果、焦糖和较低酸度。

## 冰块比例怎么调

| 杯中表现 | 调整方向 |
| --- | --- |
| 太淡 | 减少冰块或收紧粉水比 |
| 太浓 | 增加冰块或总水量 |
| 酸薄 | 调细研磨，或提高水温 |
| 苦涩 | 调粗研磨，降低水温，减少搅拌 |
| 香气弱 | 减少冰块比例，或改用更高香气的豆子 |

冰块越多，降温越快，但可用于萃取的热水越少。冰块越少，萃取更充分，但降温速度变慢，冰咖啡的清爽感会下降。

## 和冷萃的区别

| 项目 | 冰手冲 | 冷萃 |
| --- | --- | --- |
| 萃取方式 | 热水萃取 | 低温长时间浸泡 |
| 时间 | 2 到 3 分钟 | 8 到 16 小时 |
| 香气 | 更清晰、释放更快 | 更柔和、释放更慢 |
| 酸感 | 更明显 | 通常更低 |
| 适合目标 | 花香、果酸、层次 | 顺滑、低酸、提前制作 |
