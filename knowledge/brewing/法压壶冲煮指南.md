# 法压壶冲煮指南

> 法压壶是典型浸泡式萃取，操作简单、口感厚重，适合想要低门槛、稳定和多人分享的场景。

## 核心逻辑

法压壶的咖啡粉会长时间浸泡在水中，萃取主要由时间、温度、研磨和粉水比决定。金属滤网不会像滤纸一样截留油脂和细粉，所以杯中通常更厚、更浑厚，也更容易有沉淀。

法压壶不追求极高透明度。它的优势是简单、稳定、甜感和厚度容易出来；弱点是干净度不如纸滤，杯底沉淀更明显。

## 起手参数

| 参数 | 建议值 |
| --- | --- |
| 粉水比 | 1:14 到 1:16 |
| 研磨 | 中粗到粗 |
| 水温 | 90 到 96°C |
| 浸泡时间 | 4 到 6 分钟 |
| 过滤 | 缓慢下压，不要到底后反复挤压 |

第一次可用 **20g 粉 + 300g 水，94°C，浸泡 4 分钟**。如果口感薄，延长到 5 到 6 分钟或略调细；如果苦涩和沉淀多，调粗研磨并减少搅拌。

## 基础流程

1. 将咖啡粉放入法压壶。
2. 一次性注入热水，确保所有咖啡粉都被润湿。
3. 轻轻搅拌，让粉层均匀浸泡。
4. 盖上盖子但不要立刻下压，静置 4 分钟。
5. 可用勺子轻轻破开表面粉层，撇去浮沫和大颗粒。
6. 慢慢下压滤网，只压到液面附近即可，不要用力挤到底。
7. 倒出咖啡，避免长时间留在壶中继续萃取。

## 选豆思路

法压壶适合甜感、厚度和香气比较明确的豆子。中烘、中深烘、巧克力、坚果、熟果、香料调性容易得到稳定表现。浅烘水洗豆也可以用法压壶，但需要更高水温、更长浸泡和更细一点的研磨，否则容易酸薄。

日晒和厌氧豆用法压壶时要注意发酵感可能被放大。如果杯中出现酒味、闷感或尾段发苦，可以降低水温或缩短浸泡时间。

## 常见问题

| 问题 | 可能原因 | 下一次调整 |
| --- | --- | --- |
| 杯底沉淀多 | 研磨太细，或下压太用力 | 调粗研磨，缓慢下压 |
| 口感薄 | 粉水比太大、时间太短 | 收紧比例或延长浸泡 |
| 酸尖 | 萃取不足 | 升温、延长时间或略调细 |
| 苦涩 | 时间太长、研磨太细、扰动太强 | 调粗、缩短时间、减少搅拌 |
| 闷、不干净 | 细粉多、器具油脂残留 | 清洁滤网，换更均匀的研磨 |

## 和其他器具的区别

| 器具 | 主要差异 |
| --- | --- |
| V60 | V60 更干净、更依赖注水；法压更厚重、更简单 |
| 爱乐压 | 爱乐压可用纸滤压出更干净杯感；法压更适合多人分享 |
| 聪明杯 | 聪明杯是浸泡后纸滤排液，干净度高于法压 |
| 冷萃 | 冷萃也是浸泡，但低温长时间；法压是热浸泡 |
