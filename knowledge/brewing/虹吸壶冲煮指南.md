> 虹吸壶通过超细研磨 + 短时浸泡，可达到 23–24.6% 的高萃取率，同时保持杯中无油脂的干净口感。

## 核心参数

| 参数 | 数值 |
|------|------|
| 粉水比 | 1:17（偏淡可调至 1:18） |
| 研磨粒度 | **极细，接近土耳其咖啡粗细**（看起来像面粉） |
| 浸泡时间 | 总计 35 秒（前 15 秒强力搅拌 + 后 20 秒静置） |
| 目标回流时间 | 3 分钟以内 |
| 目标 TDS | ~1.5% |
| 冲煮水温 | 约 198°F（约 92°C）下浸泡 |

## 关键发现：极细研磨反而解决了问题

作者在反复尝试中发现：研磨到"中等细"时，细粉会从过滤器漏出；但研磨到**极细（面粉质感）**时，粉末反而形成致密均匀的滤层，不再泄漏——极细研磨悖论式地解决了细粉问题。

## 必要器具

| 器具 | 说明 |
|------|------|
| Hario Next 5杯虹吸壶 | 或 Technica 3杯款 |
| 光束加热器（Beam Heater） | 不能用酒精灯，功率不稳定 |
| 珠形温度探针 | 精确监控水温 |
| 能磨到极细的磨豆机 | Baratza Forté 等，普通手磨一般达不到所需细度 |
| 纸质过滤器 + 专用支架 | 替代布过滤器（维护太麻烦） |
| 竹制搅拌棒 | 搅拌用 |

## 完整冲煮流程

1. **装好纸质过滤器**，居中固定
2. **下壶注水** 400g 热水，加热至约 202°F（94°C）
3. **安装上壶**，降低加热功率至 2/3
4. **加入咖啡粉**（水沸腾上升至上壶后）
5. **搅拌阶段**：强力搅拌 15 秒，然后静置 20 秒（共 35 秒）
6. **回流**：撤火，用冷毛巾敷贴下壶底部加速回流，同时轻柔旋转上壶粉浆
7. **（可选）二次过滤**：将液体再次通过 V60 过滤，去除残余油脂

## 豆子选择

此方法**只适合发育良好、不带涩感的豆子**。作者测试有效的包括：埃塞俄比亚和肯尼亚精品豆。发育不足的豆子会出现木质感和不愉快的味道。

## 萃取率数据

- 典型萃取率：23–24.6%（渗滤计算法）
- 浓度 ~1.5% TDS
- 属于超高萃取率配方

## 维护

- 纸质过滤器：冲洗后可复用
- 布质过滤器：每次需用 OxiClean 煮沸清洁，不推荐日常使用
