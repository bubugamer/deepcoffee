# 挂耳咖啡冲泡指南

> 挂耳咖啡可以理解为“预磨粉 + 一次性滤袋”的简化手冲，优点是方便，缺点是香气和可调空间弱于现磨手冲。

## 挂耳和速溶、手冲的区别

挂耳咖啡里面是真实咖啡粉，冲泡时热水流过咖啡粉并通过滤袋过滤；速溶咖啡是已经萃取并干燥后的咖啡固体，遇水复溶；手冲则需要现磨、滤杯、滤纸和更完整的参数控制。

因此，挂耳的风味上限通常低于现磨手冲，但明显高于多数速溶咖啡。它适合办公室、旅行、没有磨豆机时使用。

## 起手参数

| 参数 | 建议值 |
| --- | --- |
| 单包粉量 | 常见 8 到 12g |
| 总注水量 | 120 到 180g |
| 粉水比 | 1:14 到 1:16 |
| 水温 | 88 到 94°C |
| 总时间 | 1:30 到 2:30 |
| 注水方式 | 少量多次，避免一次灌满 |

如果单包是 10g，第一次可以用 **150g 水**。浅烘挂耳可用 92 到 94°C；中深烘可用 88 到 92°C。

## 基础流程

1. 撕开挂耳包，轻轻抖平咖啡粉。
2. 展开两侧纸耳，稳定挂在杯口。
3. 先注入少量热水润湿咖啡粉，等待 20 到 30 秒。
4. 分 2 到 4 次缓慢注水，不要让水位长期淹过滤袋顶部。
5. 达到目标水量后，取下挂耳包。
6. 轻轻摇匀杯中咖啡后饮用。

## 怎么判断挂耳是否适合自己

| 需求 | 适合程度 |
| --- | --- |
| 想省事、无需磨豆机 | 很适合 |
| 想在办公室稳定喝黑咖啡 | 很适合 |
| 想练习注水和萃取控制 | 只能作为入门 |
| 追求花香、层次和极高新鲜度 | 不如现磨手冲 |
| 想长期记录和微调参数 | 不如手冲或爱乐压 |

挂耳最重要的是新鲜度和密封保存。预磨粉接触氧气后香气流失更快，开封后应立刻冲泡。

## 常见问题

| 问题 | 可能原因 | 下一次调整 |
| --- | --- | --- |
| 太淡 | 水太多，或注水太快 | 减少水量，分段慢注 |
| 太苦 | 水温太高，浸泡太久 | 降低水温，缩短总时间 |
| 酸薄 | 水温太低或萃取不足 | 升温，延长闷蒸和总时间 |
| 有纸味 | 包材或杯子影响 | 先用少量热水润湿滤袋并倒掉 |
| 香气弱 | 预磨粉放置过久 | 选择更新鲜包装，开封即冲 |

## 可不可以做冰挂耳

可以。做法类似冰手冲：杯中先放冰块，热水注入挂耳包完成萃取，咖啡液滴到冰块上快速降温。由于热水量变少，建议比热挂耳略减少冰块或略提高萃取浓度。

示例：10g 挂耳粉，杯中放 60g 冰块，用 90 到 110g 热水冲泡。喝起来太淡时减少冰块或热水；太浓时补冰或补水。
