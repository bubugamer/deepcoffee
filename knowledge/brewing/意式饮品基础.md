# 意式饮品基础

> 意式饮品的核心是浓缩咖啡。美式、拿铁、澳白、卡布奇诺等差异，主要来自出液比例、加水、加奶和奶泡厚度。

## Espresso、Ristretto 和 Lungo

| 饮品 | 常见比例 | 杯中特点 |
| --- | --- | --- |
| Ristretto | 1:1 到 1:1.5 | 更浓、更厚，酸甜集中，尾段萃取少 |
| Espresso | 1:2 左右 | 意式标准基底，平衡浓度和萃取 |
| Lungo | 1:3 以上 | 出液更多，浓度下降，后段苦感风险更高 |

这里的比例指“粉量 : 出液量”。例如 18g 粉出 36g 液体，就是约 1:2。Ristretto 不是简单“更小杯的 Espresso”，而是减少后段出液，让浓度和前段风味更集中。

## 美式咖啡

美式是浓缩咖啡加水。常见做法是先萃取一份 Espresso，再加入热水或冰水。它的风味取决于浓缩本身、加水比例和水温。

| 类型 | 起手比例 |
| --- | --- |
| 热美式 | 1 份 Espresso + 2 到 4 倍热水 |
| 冰美式 | 冰块 + 冷水 + Espresso |
| 长黑 | 先加热水，再倒入 Espresso，保留更多 crema |

如果美式喝起来苦，先检查浓缩是否过萃，不要只靠加水掩盖。加水只能降低浓度，不能消除萃取问题。

## 拿铁、澳白、卡布奇诺

| 饮品 | 核心差异 | 杯中表现 |
| --- | --- | --- |
| 拿铁 | 牛奶量较多，奶泡薄 | 柔和、甜感明显、咖啡感较低 |
| 澳白 | 杯量更小，奶泡薄，咖啡比例更高 | 咖啡感更集中，口感细腻 |
| 卡布奇诺 | 奶泡更厚 | 口感蓬松，咖啡和奶泡层次更明显 |

这些饮品没有全球完全统一的杯量标准。判断它们时，与其只看名字，不如看浓缩份数、总杯量、牛奶量和奶泡厚度。

## 意式选豆思路

意式萃取压力高、浓度高，会放大豆子的酸、苦、涩和烘焙缺陷。新手更容易从中烘到中深烘、甜感和厚度明确的豆子开始。常见方向：

- 拼配豆：稳定、容易有厚度和甜感，适合奶咖和日常出品。
- SOE：产地风味更明确，但对萃取参数和豆子状态更敏感。
- 深烘豆：低酸、厚重、适合奶咖，但焦苦风险更高。
- 浅烘豆：花果香明显，但容易酸尖，需要更好的设备和参数控制。

## 基础萃取参数

| 参数 | 起手范围 |
| --- | --- |
| 粉量 | 16 到 20g |
| 出液量 | 粉量的 1.8 到 2.5 倍 |
| 时间 | 25 到 35 秒 |
| 研磨 | 细研磨，按流速微调 |
| 目标 | 出液稳定、甜感清晰、苦涩不过度 |

如果流速太快、酸薄，通常先调细研磨；如果流速太慢、苦涩，通常先调粗研磨。每次只改一个变量，避免无法判断原因。

## 和摩卡壶的区别

摩卡壶能做出浓郁咖啡，但压力、温度控制和粉饼结构都不同于意式机。它可以作为家用浓缩替代，但不能完全等同于 Espresso。用摩卡壶做奶咖时，重点是降低焦苦、控制浓度，而不是追求意式机式 crema。
