# Project Origin CM Selections

Project Origin 的 CM Selections 是以二氧化碳浸渍为核心的生豆选择项目，常出现在特殊处理法、竞赛批次和高辨识度风味型咖啡语境中。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 所属机构 | Project Origin |
| 产品类型 | 生豆选择项目 / 处理法项目 |
| 常见关联 | Carbonic Maceration、CMW、CMN、特殊发酵批次 |
| 识别价值 | 帮助系统把 “CM Selections” 识别为生豆商产品线，而不是烘焙商产品 |
| 主要来源 | Project Origin 官方 CM Selections 页面 |

## 怎么看

- 它描述的是 Project Origin 流通和开发的生豆处理项目。
- 具体批次仍需要结合产地、生产者、年份和烘焙商页面判断。
