# Sucafina IMPACT

IMPACT 是 Sucafina 的责任采购项目，用于描述其围绕农户韧性、土壤健康、生物多样性和供应链协作的生豆采购体系。

| 项目 | 说明 |
| --- | --- |
| 所属机构 | Sucafina |
| 产品类型 | 责任采购项目 |
| 常见关联 | responsibly sourced coffee、IMPACT Beyond、supply chain program |
| 识别价值 | 帮助系统把 “IMPACT” 识别为 Sucafina 的采购项目语境 |
| 主要来源 | Sucafina 官方 IMPACT 页面 |

## 怎么看

- 它不是单个庄园或具体熟豆产品，而是生豆采购项目。
- 具体豆卡仍应记录产地、生产者、批次、品种和处理法。
