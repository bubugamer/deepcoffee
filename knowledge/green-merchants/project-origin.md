# Project Origin

寻豆机构与精品生豆贸易来源，不与庄园/生产者混用。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 机构类型 | 寻豆机构 / 生豆进口商 |
| 总部/主要市场 | Australia / Canberra 语境 |
| 角色 | 绿色咖啡采购、进口、CM Selections 和生产者协作 |
| 生豆来源覆盖 | 全球多产区，强调 CM 处理项目 |
| 服务方式 | 生豆贸易、项目合作、处理法研发与批次选择 |
| 与生产者关系 | 与生产者协作开发 CM 等处理批次 |

## 怎么看

- 用于识别生豆贸易、进口、出口或寻豆渠道。
- 它描述咖啡如何流通，不等同于庄园、生产者或处理站。
