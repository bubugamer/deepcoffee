# Coffee Buff

Coffee Buff（加福咖啡）是由 2020 年世界咖啡冲煮大赛（CBrC）中国区冠军李震，与曾任世界咖啡赛事评审的杨逸凡共同创立的精品咖啡品牌，以专业烘焙、冠军级冲煮配方以及极具分享性的咖啡教学服务在业内闻名。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 国家/地区 | China |
| 品牌类型 | 独立精品烘焙商 |
| 创始人 | 李震（2020 CBrC 中国区冠军）、杨逸凡（赛事评审） |
| 主要产品方向 | 线上精品熟豆烘焙、专业咖啡培训，后开设线下实体店 |
| 特色 | 冠军级冲煮手法与配方；创新风味（如夹心水果糖风味冰咖） |
| 发展动向 | 在上海积累人气后入驻青岛，扩大空间并完成产品全面升级 |
| 官方渠道 | 品牌自有渠道 |

## 怎么看

- 用于识别熟豆品牌、烘焙商或连锁咖啡品牌。
- 连锁品牌和独立烘焙厂牌已用品牌类型区分。
