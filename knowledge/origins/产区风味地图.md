# 产区风味地图

产区不能直接决定风味，但它能提供第一层预期：海拔、气候、常见品种、处理传统和市场分级，会共同影响一支豆的表现。

## 快速读产区

| 产区 | 常见印象 | 阅读重点 |
| --- | --- | --- |
| 埃塞俄比亚 | 花香、柑橘、茶感、复杂果香 | 继续看耶加雪菲、西达摩、古吉等更小地名 |
| 肯尼亚 | 明亮酸质、莓果、黑加仑、饱满结构 | 注意 AA、AB、处理站、合作社和拍卖体系 |
| 巴拿马 | 瑰夏、高端庄园、竞赛批次 | 重点看庄园、品种和 Best of Panama 语境 |
| 哥伦比亚 | 产区跨度大，风格从坚果甜感到花果香都有 | 需要看具体省份、庄园和处理法 |
| 巴西 | 坚果、巧克力、甜感、拼配基础 | 注意 Cerrado、Minas Gerais 等地区和日晒/去果肉日晒 |
| 危地马拉 | 火山土壤、高海拔、明亮酸质和可可感 | 安提瓜、薇薇特南果等区域差异明显 |
| 哥斯达黎加 | 蜜处理、微处理厂、甜感和干净度 | 关注 Tarrazu、West Valley 等官方产区 |
| 中国云南 | 国内精品咖啡核心产地，常见水洗、日晒和厌氧 | 需要结合烘焙商、庄园和批次信息阅读 |

## 不要只靠国家判断

同一个国家内部差异很大。埃塞俄比亚的耶加雪菲和古吉不应混成一种风味；哥伦比亚不同省份、海拔和处理法也会差异明显。看豆袋时，国家只是入口，真正有价值的是“国家 + 子产区 + 来源主体 + 品种 + 处理法”。
