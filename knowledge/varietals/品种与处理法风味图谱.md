# 品种与处理法风味图谱

品种决定咖啡树的遗传基础，处理法决定采收后的风味表达方式。读一支豆时，不要只看“瑰夏”或“厌氧”，要把品种、产区、处理法和烘焙程度放在一起判断。

## 常见品种怎么建立预期

| 品种 | 常见风味预期 | 阅读提醒 |
| --- | --- | --- |
| 瑰夏 / Gesha | 花香、柑橘、茶感、热带水果 | 高关注品种，但不同产区和处理法差异很大 |
| 波旁 / Bourbon | 甜感、坚果、圆润口感 | 是许多现代品种的重要基础 |
| 铁皮卡 / Typica | 干净、平衡、传统阿拉比卡风格 | 常作为历史品种理解入口 |
| SL28 / SL34 | 明亮酸质、莓果、黑加仑 | 常和肯尼亚产区一起出现 |
| 帕卡玛拉 | 大颗粒、复杂度高、可能有花果香 | 需结合产地和处理法判断 |
| 卡杜拉 / 卡杜艾 | 拉美常见，表现稳定 | 风味更依赖产区、处理法和烘焙 |
| 粉红波旁、Sidra、Mejorado | 精品市场高关注名称 | 名称需要来源核验，不要只按市场宣传判断 |

## 处理法怎么改变表达

水洗通常让品种和产区特征更清楚；日晒会加强果香、甜感和口感厚度；蜜处理介于两者之间；厌氧、CM、热冲击等现代处理法会放大水果感、酒香或发酵感，也更需要看生产者是否清楚披露流程。

同一个品种经过不同处理，杯中表现可能相差很大。瑰夏水洗可能轻盈、花香明显；瑰夏日晒可能更甜、更热带水果；瑰夏厌氧则可能更集中、更有发酵调性。

## 对选豆和冲煮有什么帮助

如果你想要花香和清晰度，可以优先看水洗瑰夏、埃塞原生种和高海拔水洗豆；如果你想要甜感和果汁感，可以看日晒、蜜处理或发酵控制做得干净的批次；如果你怕发酵味，看到厌氧、共发酵或热冲击时，应先查看烘焙商描述和用户评价。

冲煮上，花香型和水洗豆通常要避免萃取不足；强发酵型豆则要优先控制涩感、杂味和发酵感过冲。
