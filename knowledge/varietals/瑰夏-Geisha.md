# 瑰夏（Geisha / Gesha）

瑰夏是精品咖啡界最具代表性的品种之一，以独特的茉莉花香、桃子、芒果等热带水果风味著称，价格高昂，竞拍纪录达一万美金一磅（2023年）。

| 品种类型 | 阿拉比卡品种 |
| 来源/发现地 | 埃塞俄比亚 Gesha 相关材料 |
| 亲本或谱系 | T2722 等传播材料 |
| 常见产区 | 巴拿马、哥伦比亚、危地马拉、埃塞俄比亚 |
| 典型风味 | 茉莉花、柑橘、茶感、热带水果 |
