# 卡杜隆 / Caturron

Caturron（卡杜隆）是近年来在哥伦比亚乌伊拉（Huila）产区发现的较为罕见、神秘的咖啡品种，与 Caturra（卡杜拉）相关但是不同的品种。名字在当地通常带有「大卡杜拉（Big Caturra）」之意，因其果实和豆粒比普通卡杜拉明显更大。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 标准名 | Caturron |
| 常见别名 | Caturron、Caturrón、卡杜隆 |
| 品种类型 | 自然杂交/突变种（与 Caturra 相关但不同） |
| 来源/发现地 | 哥伦比亚 乌伊拉（Huila） |
| 亲本或谱系 | DNA 测试显示可能为 Timor 杂交种与波旁/卡杜拉的自然杂交或突变后代 |
| 农艺特性 | 叶锈病抗性极佳、产量表现优异 |
| 典型风味 | 奔放多变：莓果、热带水果（芒果、凤梨、西瓜）+ 独特草本植物香气，辨识度高 |

## 怎么看

- 常见于豆袋标签中的品种栏。
- Caturron 与 Caturra 是两个不同品种，不要混为一谈。
