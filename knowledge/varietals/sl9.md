# SL9 / SL-09

SL9 / SL-09 是肯尼亚 Scott Agricultural Laboratories（SL）选育系列中的少见品系，与更知名的 SL28、SL34 同属 SL 系列。近年在秘鲁 Incahuasi Valley 一带的咖啡中，SL-09、Gesha Inca、Inca Gesha、Rare Variety 等名称容易混用，需要谨慎解读。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 标准名 | SL9 / SL-09 |
| 常见别名 | SL9、SL9*、SL-09、SL09、Gesha Inca、Inca Gesha |
| 品种类型 | 单株选育系 |
| 来源/发现地 | 肯尼亚 Scott Agricultural Laboratories |
| 亲本或谱系 | Scott Labs 选育；部分商业样本可能与 Ethiopian Legacy / SL 系遗传材料相近 |
| 常见产区 | 肯尼亚少见；近年在秘鲁 Incahuasi Valley 商业流通中更常被讨论 |
| 典型风味 | 花香、柑橘、优雅酸质等描述较常见，但需结合产区、处理法和具体批次 |

## 怎么看

- 常见于豆袋标签中的品种栏。
- SL 系列指肯尼亚 Scott Labs 选育的不同品系，编号不同即不同品系。
- 标注为 Gesha Inca 或 Inca Gesha 的秘鲁批次，不应直接等同于 Geisha。
- 如果豆袋或烘焙商写作 Rare Variety，通常表示该样本无法与数据库中的标准指纹完全对应，但可能仍与 SL-09 相近。
- 对用户来说，最重要的是把豆袋标注原样保存，并把 SL-09 / SL9 / SL9* 作为同义检索词处理。

## 与 Geisha 的关系

秘鲁 Incahuasi Valley 一带曾因花香、柑橘调和优雅酸质，被当地或市场称作 Gesha Inca / Inca Gesha。后续基因检测显示，这些样本与 Geisha 并不能简单划等号，更接近 SL-09 或 SL-09 相关遗传材料。

需要区分三层信息：

1. **市场名称**：Gesha Inca / Inca Gesha 可能是商业或地方名称。
2. **品种判断**：检测结果可能指向 SL-09 或与 SL-09 相近的稀有遗传指纹。
3. **杯中风味**：风味像 Geisha，不代表遗传上就是 Geisha。

因此，遇到这类豆袋时，建议保留烘焙商原文，同时把它理解为存在品种命名争议的标注，而不是直接改写成 Geisha。

## 豆袋记录建议

- 品种名称可以按豆袋原文记录，例如 `SL9*`、`SL-09` 或 `Gesha Inca`。
- 阅读和检索时，可以把 `SL9`、`SL-09`、`SL09`、`SL9*` 理解为同一组常见写法。
- 不建议自动把 `Gesha Inca` 改写成 `Geisha`；除非豆袋明确标为 Geisha 且来源可信。
- 如果需要解释给他人听，可以说：“秘鲁常见的 SL-09 / Gesha Inca 相关标注，风味可能接近 Geisha，但品种身份不应直接等同于 Geisha。”
