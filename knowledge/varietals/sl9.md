# SL9

SL9 是肯尼亚 Scott Agricultural Laboratories（SL）选育系列中的一个品系，与更知名的 SL28、SL34 同属该机构早期的单株选育成果。相对 SL28/SL34 较为少见，偶见于肯尼亚及部分庄园批次的品种标注（也写作 SL9*）。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 标准名 | SL9 |
| 常见别名 | SL9、SL9* |
| 品种类型 | 单株选育系 |
| 来源/发现地 | 肯尼亚 Scott Agricultural Laboratories |
| 亲本或谱系 | Scott Labs 选育（与 SL28/SL34 同源机构） |
| 常见产区 | 肯尼亚 |
| 典型风味 | 视产区与处理法而定 |

## 怎么看

- 常见于豆袋标签中的品种栏。
- SL 系列指肯尼亚 Scott Labs 选育的不同品系，编号不同即不同品系。
