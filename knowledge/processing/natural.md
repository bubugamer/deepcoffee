# 日晒处理 / Natural Process

整果干燥处理法，常与明显果香、发酵感和较高甜感关联。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 日晒处理 |
| 英文名 | Natural Process |
| 处理类别 | 传统处理法 |
| 核心步骤 | 整果干燥后脱壳 |
| 发酵/干燥环境 | 开放或受控干燥环境 |
| 常见风味影响 | 增强果香、甜感和发酵调性 |
| 常见产区 | 埃塞俄比亚、巴西、拉美等产区 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
