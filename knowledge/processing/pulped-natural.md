# 去果肉日晒 / Pulped Natural

去除果皮后带果胶干燥，是巴西等产地常见处理方式。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 去果肉日晒 |
| 英文名 | Pulped Natural |
| 处理类别 | 传统处理法 |
| 核心步骤 | 去皮后带果胶干燥 |
| 发酵/干燥环境 | 干燥为核心步骤 |
| 常见风味影响 | 强化甜感和坚果/巧克力调性 |
| 常见产区 | 巴西等产区 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
