# 湿刨法 / Wet-hulled

印度尼西亚常见处理方式，常与独特草本、香料和厚重口感关联。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 湿刨法 |
| 英文名 | Wet-hulled |
| 处理类别 | 产地特色处理法 |
| 核心步骤 | 含水率较高时脱壳后继续干燥 |
| 发酵/干燥环境 | 湿润环境与分段干燥 |
| 常见风味影响 | 草本、香料感和厚重口感 |
| 常见产区 | 印度尼西亚 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
