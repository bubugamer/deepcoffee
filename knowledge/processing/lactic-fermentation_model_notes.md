# 乳酸发酵模型注释

## 识别边界

- `Lactic fermentation` / `乳酸发酵` 通常描述发酵方向或微生物控制，不等于所有厌氧处理。
- 豆袋写乳酸发酵时，不要自动改写成普通厌氧或共发酵。
- 如果文本没有说明基础处理法，保留 `Lactic fermentation` 原文。

## 回答策略

- 可解释其常见目标是提升圆润酸质、甜感或发酵层次，但不要保证杯中一定有乳酸味。
