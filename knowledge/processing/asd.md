# ASD 处理 / Anaerobic Slow Dry

慢速干燥与厌氧语境相关的处理名，通常用于描述厌氧发酵后拉长干燥节奏的处理方式。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | ASD 处理 |
| 英文名 | Anaerobic Slow Dry |
| 处理类别 | 现代处理法 |
| 核心步骤 | 厌氧语境结合慢速干燥 |
| 发酵/干燥环境 | 密闭发酵与慢速干燥 |
| 常见风味影响 | 提升甜感、发酵感和浓郁度 |
| 常见产区 | 多见于现代精品咖啡批次，具体以豆袋或产地资料为准 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
