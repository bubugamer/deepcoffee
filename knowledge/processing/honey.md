# 蜜处理 / Honey Process

保留不同程度果胶后干燥，常用于描述介于水洗和日晒之间的风味表现。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 蜜处理 |
| 英文名 | Honey Process |
| 处理类别 | 传统/现代混合处理法 |
| 核心步骤 | 去皮后保留部分果胶干燥 |
| 发酵/干燥环境 | 果胶参与干燥过程 |
| 常见风味影响 | 增强甜感、body 和果香 |
| 常见产区 | 哥斯达黎加、中美洲等产区 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
