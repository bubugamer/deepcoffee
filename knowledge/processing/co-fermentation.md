# 共发酵 / Co-fermentation

与非咖啡材料共同发酵的争议性处理类别，阅读时需要结合来源理解。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 共发酵 |
| 英文名 | Co-fermentation |
| 处理类别 | 现代争议处理类别 |
| 核心步骤 | 咖啡与非咖啡材料共同发酵 |
| 发酵/干燥环境 | 添加物参与发酵环境 |
| 常见风味影响 | 产生强识别度外源风味 |
| 常见产区 | 多见于现代精品咖啡批次，具体以豆袋或产地资料为准 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
