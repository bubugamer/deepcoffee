# 半水洗 / Semi-washed

介于传统水洗和干燥流程之间的处理表述，需要在资料中注明产地语境。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 半水洗 |
| 英文名 | Semi-washed |
| 处理类别 | 处理法表述 |
| 核心步骤 | 介于水洗与干燥流程之间 |
| 发酵/干燥环境 | 按产地语境确认 |
| 常见风味影响 | 兼具一定干净度和厚度 |
| 常见产区 | 亚洲、拉美部分产区 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
