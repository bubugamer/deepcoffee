# 酵母接种发酵 / Yeast Inoculated

加入特定酵母或微生物进行控制发酵的处理类别。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 酵母接种发酵 |
| 英文名 | Yeast Inoculated |
| 处理类别 | 现代发酵处理法 |
| 核心步骤 | 加入特定酵母后控制发酵 |
| 发酵/干燥环境 | 微生物参与的受控发酵 |
| 常见风味影响 | 提升目标风味和稳定性 |
| 常见产区 | 多见于现代精品咖啡批次，具体以豆袋或产地资料为准 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
