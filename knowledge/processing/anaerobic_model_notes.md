# 厌氧处理模型注释

## 识别边界

- `厌氧` / `Anaerobic` 是处理环境或大类标签，不等同于完整处理法组合。
- `厌氧日晒`、`厌氧水洗`、`厌氧蜜处理` 应保留组合描述，不要只保存成 `厌氧`。
- 如果豆袋只写 `Anaerobic`，处理法可保存原文，不要补推日晒、水洗或蜜处理。

## 回答策略

- 回答风味时说明厌氧常与发酵感、熟果或酒感相关，但实际表现取决于基础处理法和发酵控制。
