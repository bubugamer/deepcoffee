# 二氧化碳浸渍 / Carbonic Maceration

借鉴葡萄酒的处理概念，Project Origin 的 CM 资料是主要来源。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 二氧化碳浸渍 |
| 英文名 | Carbonic Maceration |
| 处理类别 | 现代发酵处理法 |
| 核心步骤 | 成熟咖啡樱桃经分选后进入充入二氧化碳的密闭罐 |
| 发酵/干燥环境 | 二氧化碳参与的低氧环境 |
| 常见风味影响 | 通过控制酵母和发酵环境塑造更明确的果香与层次 |
| 常见产区 | Project Origin CM 产地 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
