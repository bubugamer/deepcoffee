# 水洗处理 / Washed Process

去除果肉后发酵清洗再干燥，常用于突出清晰酸质和干净度。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 水洗处理 |
| 英文名 | Washed Process |
| 处理类别 | 传统处理法 |
| 核心步骤 | 去果肉、发酵、清洗、干燥 |
| 发酵/干燥环境 | 有水发酵或清洗流程 |
| 常见风味影响 | 提升干净度与清晰酸质 |
| 常见产区 | 拉美、东非等多个产区 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
