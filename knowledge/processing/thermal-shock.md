# 热冲击处理 / Thermal Shock

以温度变化作为处理变量的现代实验处理法。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 热冲击处理 |
| 英文名 | Thermal Shock |
| 处理类别 | 现代实验处理法 |
| 核心步骤 | 以温度变化作为处理变量 |
| 发酵/干燥环境 | 温度控制环境 |
| 常见风味影响 | 强化香气释放和风味辨识度 |
| 常见产区 | 多见于现代精品咖啡批次，具体以豆袋或产地资料为准 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
