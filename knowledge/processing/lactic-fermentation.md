# 乳酸发酵 / Lactic Fermentation

围绕乳酸菌或乳酸发酵结果描述的处理类别。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 乳酸发酵 |
| 英文名 | Lactic Fermentation |
| 处理类别 | 现代发酵处理法 |
| 核心步骤 | 围绕乳酸菌或乳酸发酵结果控制流程 |
| 发酵/干燥环境 | 微生物参与的受控发酵 |
| 常见风味影响 | 提升酸质圆润度和乳酸调性 |
| 常见产区 | 多见于现代精品咖啡批次，具体以豆袋或产地资料为准 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
