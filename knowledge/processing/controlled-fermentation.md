# 控制发酵 / Controlled Fermentation

强调温度、时间、容器和微生物控制的处理类别。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 控制发酵 |
| 英文名 | Controlled Fermentation |
| 处理类别 | 现代处理类别 |
| 核心步骤 | 控制时间、温度、容器和微生物变量 |
| 发酵/干燥环境 | 受控发酵环境 |
| 常见风味影响 | 提升风味稳定性和目标表达 |
| 常见产区 | 多见于现代精品咖啡批次，具体以豆袋或产地资料为准 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
