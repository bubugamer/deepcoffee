# 延长发酵 / Extended Fermentation

延长发酵时间以改变风味结构的处理类别。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 延长发酵 |
| 英文名 | Extended Fermentation |
| 处理类别 | 现代发酵处理法 |
| 核心步骤 | 延长发酵时间后干燥 |
| 发酵/干燥环境 | 时间控制为核心变量 |
| 常见风味影响 | 提升复杂度、果香和发酵感 |
| 常见产区 | 多见于现代精品咖啡批次，具体以豆袋或产地资料为准 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
