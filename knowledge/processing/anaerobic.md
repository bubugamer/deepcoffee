# 厌氧发酵 / Anaerobic Fermentation

在低氧或密闭环境中进行发酵，常见于现代精品处理法。

## 快速了解

| 项目 | 说明 |
| --- | --- |
| 中文名 | 厌氧发酵 |
| 英文名 | Anaerobic Fermentation |
| 处理类别 | 现代发酵处理法 |
| 核心步骤 | 密闭或低氧环境中发酵后干燥 |
| 发酵/干燥环境 | 低氧/密闭发酵环境 |
| 常见风味影响 | 提升发酵感、果香和复杂度 |
| 常见产区 | 哥伦比亚、巴拿马、中美洲等产区 |

## 怎么看

- 用于理解咖啡从采收到干燥之间的处理路径。
- 处理法会影响香气、酸质、甜感、发酵感和干净度。
