# 家用意式机选购指南

## 核心选购框架（三步筛选法）

### 第一步：确认预算

| 预算区间 | 建议 |
|---------|------|
| ¥2000 以下 | **直接跳过意式机**，买摩卡壶或专注手冲 |
| ¥2000–5000 | 一体机（含研磨），适合中深烘中度探索者 |
| ¥5000–15000 | 半自动机 + 独立磨豆机，可玩性更高 |
| ¥15000–30000 | Lelit V3、部分高端国产等，旋转泵/齿轮泵，变压能力 |
| ¥30000+ | Decent、E61机头商用机，顶级玩家 |

**¥2000以下为何不推荐**：
- 基本都是玩具机，萃取稳定性差、蒸汽功率不足
- 即使专业咖啡师使用也难出好杯子
- 不如把这笔钱用在摩卡壶（¥200–400）+ 好豆子上

### 第二步：判断使用习惯

- **懒人 / 追求便利**：全自动咖啡机、胶囊咖啡机
- **只喝中深烘奶咖**：一体机（含内置锥刀磨豆机），3000–5000元段位差异不大
- **愿意折腾、有探索欲**：半自动 + 独立磨豆机

**特别提醒**：买咖啡机前想想"清洁成本"——咖啡粉会跑到意想不到的地方，机器清理是日常功课。

### 第三步：选具体参数

## 核心参数解析

### 加热系统

| 类型 | 特点 | 适合场景 |
|------|------|---------|
| 加热块（即热式） | 开机快（3分钟），单次用水量小，水更新鲜 | 一天1–2杯的家用 |
| 单锅炉 | 蒸汽/萃取共用，无法同时操作 | 入门家用 |
| 子母锅炉 | 小锅炉萃取+大锅炉蒸汽，两者联动 | 中端家用 |
| 双锅炉 | 独立调节萃取温度和蒸汽温度 | 发烧玩家 |

**注意**：锅炉≠更好。一天喝1–2杯的用户，1升锅炉需要一周才能喝完 → 反复加热，不如加热块。Decent（¥3万+）也用加热块，水温精准度靠算法控制。

### 水泵类型

| 类型 | 特点 |
|------|------|
| 震动泵 | 家用机最常见，体积小造价低，有脉冲式压力波动，Decent也用此 |
| 旋转泵 | 商用机主流，安静，压力稳定 |
| 齿轮泵 | 变压能力强，机械感强（声音像引擎），多见于发烧机 |

**结论**：¥5000以下基本都是震动泵，不用纠结。

### 变压能力（Pressure Profiling）

- 能在萃取过程中动态调整压力，影响风味走向
- 顶级机器（Decent等）可精细设置压力曲线
- 国产品牌（惠家等）在中端价位也开始提供变压功能

## 一体机 vs 半自动机

| | 一体机 | 半自动机 |
|--|--------|---------|
| 内置磨豆机 | 有（锥刀为主） | 无 |
| 适合豆子 | 中深烘（容错率高） | 浅烘也适合 |
| 上手难度 | 低 | 较高 |
| 玩法空间 | 有限 | 大 |
| 典型场景 | 早上快速出一杯拿铁 | 认真研究萃取参数 |

**一体机关键提示**：同价位一体机研磨+萃取差异不大，主要差在外观、细节功能（蒸汽效率、蒸汽探温功能、上下水等）。

## 代表机型参考

| 机型 | 价格区间 | 特点 |
|------|---------|------|
| 摩卡壶 | ¥200–400 | 无需专业知识，也能做高浓度咖啡 |
| 一体机（伯父/百胜图等） | ¥3000–5000 | 中深烘首选，上手简单 |
| Lelit V3 | ¥8000–12000 | 工业感外观，旋转泵，被动变压，性价比稳 |
| 国产中高端（惠家等） | ¥6000–10000 | 近年快速发展，功能媲美进口 |
| Decent DE1 | ¥25000–35000 | 终极参数控制，触屏界面，中国大陆首台用户为微胖 |

## 关于家用咖啡机的常见误区

1. **锅炉一定比加热块高级**：错，Decent用加热块做到顶级温控
2. **越贵越好喝**：错，赛车手开普通车也能跑很好；设备是上限，技术是决定因素
3. **2000以下有宝藏机器**：谨慎，这个价位95%以上是玩具机，多见于网红博主广告
4. **五千以下要选锅炉**：不必要，加热块在家用场景完全够用
