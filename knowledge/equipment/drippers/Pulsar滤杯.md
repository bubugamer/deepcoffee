> Pulsar 是专为深粉床和浸泡式手冲设计的圆柱形滤杯，由天体物理学家 Jonathan Gagné 与 NextLevel Brewer 联合开发，带阀门可控流速。

## 设计理念

作者在《The Physics of Filter Coffee》中提出了理想滤杯的设计要求：

- **圆柱形**：避免水沿锥壁绕过粉床（V60 的根本缺陷）
- **最大开放滤纸面积**：减少底部过滤层对流速的限制
- **带阀门**：可控制出液时机，支持浸泡式 / 半浸泡式 / 渗滤式切换
- **分散网（Dispersion Screen）**：均匀分散注水，避免局部搅动导致细粉堵塞

名称"Pulsar"来自作者天体物理学家的身份，同时也暗示"脉冲式注水（pulse）"的手法。

## Pulsar vs Pulsar Mini

| 指标 | Pulsar | Pulsar Mini |
|------|--------|-------------|
| 适合粉量 | 20–30g | 12–20g |
| 粉床深度（15g） | 偏浅（<10mm） | 约 17mm（最优） |
| 推出原因 | 原版 | 粉床过浅问题 |

Pulsar Mini 是应读者需求推出的缩小版，专为喜欢 12–18g 小剂量的用户优化粉床深度。

## 推荐冲煮配方（浸泡-渗滤混合）

| 参数 | 数值 |
|------|------|
| 粉量 | 20–30g（Mini：12–20g） |
| 粉水比 | 1:17（偏浓可用 1:16） |
| 研磨 | ~800 微米（比 V60 略粗） |
| 总时长 | 3:30–4:30 |
| 水温 | 99°C |

**步骤**：
1. 底部预注水，插入干燥滤纸，开阀让水过滤器流下固定滤纸
2. 加入磨好的咖啡，用分散盖刮平
3. 注入粉重 3 倍的水进行闷蒸；水开始滴出时关闭阀门
4. 轻柔旋转壶体使粉床均匀
5. 1 分钟时，脉冲式注水维持约 1cm 水面高度
6. 持续注水至目标水量
7. 保持稳定水位减少温度波动，降低涩感风险

## 风味倾向

- **长闷蒸（60s+）**：甜感、果酱感、饱满
- **短闷蒸（15s，类 V60 模式）**：茶感、酸质鲜明、干净

深烘焙豆使用长闷蒸风险较高，容易出现杂味。

## 粉床深度：最关键的变量

作者在 Pulsar Mini 文章中系统总结了**粉床深度**对冲煮的四重影响：

| 影响 | 说明 |
|------|------|
| 流速与研磨 | 粉床越深，阻力越大，需要更粗研磨或更长时间 |
| 抗干扰性 | 粉床越厚，局部扰动（通道）的比例影响越小，更宽容 |
| 过滤质量 | 粉床越深，细粉和涩感物质被截留越多 |
| 萃取动态 | 底层粉接触到的水已含较多溶质，萃取梯度不同 |

**粉床深度参考参数**（EG-1 磨豆机，1:17 粉水比）：

| 粉床深度 | 参考粉量 | 冲煮时间 |
|---------|---------|---------|
| 浅（10mm） | ~12g | ~3:00 |
| 中（17mm） | ~15g | ~3:45 |
| 深（22mm） | ~20g | 4:30+ |

**重要认知**：改变滤杯直径（但保持粉床深度不变）对风味几乎没有影响，直径主要决定你能冲多少量，而非如何冲。

## 维护

- 各部件可拆洗，耐洗碗机（部分清洁剂可能影响风味，建议清水）
- 筒内壁油脂需特别冲洗
