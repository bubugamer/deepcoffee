> Stagg [X] 是平底宽口滤杯，保温性和萃取均匀性优于 V60，但需要加装金属网防止滤纸塌陷，重复性略差。

## 核心优势

### 1. 更好的保温性
Stagg [X] 腔体温度比 V60 更高且更稳定，冲煮过程中粉浆温度波动更小。温度稳定 → 化学萃取更一致。

### 2. 风味更甜、酸感更柔
对比使用相同豆子、相同参数，Stagg [X] 出品比 V60"更甜、更少酸"。作者认为原因是：
- 更优秀的保温性
- 平底设计减少了绕流效应——V60 的锥形会让水沿壁流下，绕过粉床

### 3. 平底消除绕流
锥形滤杯（V60）中，水会沿最小阻力路径流到锥尖，而非均匀通过粉床。平底让水必须竖直穿透粉层。

## 主要技术挑战

### 滤纸塌陷问题
平底滤杯的滤纸在底部没有结构支撑，容易贴合底部、失去过滤面积，导致严重堵塞。

**解决方案：在底部放置金属网**

| 网型 | 效果 |
|------|------|
| 平面茶叶过滤网 | 基础防塌，可用 |
| 尖刺型网 | 增加滤纸展开面积，但放置难以重复 |
| 粗目意面过滤网 | 作者当前最优选，与两种滤纸都兼容 |

## 推荐冲煮参数

| 参数 | 数值 |
|------|------|
| 粉量 | 22–26g（推荐 25g） |
| 粉水比 | 1:17（偏好） |
| 水温 | 99°C（浅烘）/ 96°C（深烘） |
| 注水段数 | 4 段（V60 通常 2 段），每段约 90g |
| 目标冲煮时长 | 3:45–5:00 |

**注水手法**：
- 粉床中心挖坑（巢状），第一段水注入坑内
- 闷蒸注水量：粉重 × 2.5–3.5
- 闷蒸完成后轻旋（0.5 秒）；闷蒸本身旋转 2–3 秒
- 每段水面下降约 1 英寸后再注下一段

## 滤纸选择

| 选项 | 优劣 |
|------|------|
| Stagg 原装折叠滤纸 | 厚、纸味重，需充分预润湿 |
| V60 滤纸塞入 Stagg | 孔隙质量更好、厚度适宜，但折叠麻烦、一致性略差 |

V60 滤纸塞入方法：用水龙头水压将滤纸冲成适合 Stagg 底部的形状，然后贴壁固定。

## 重复性的取舍

| 指标 | V60 | Stagg [X] |
|------|-----|-----------|
| 萃取率一致性 | ±0.2% | ±0.3–0.4% |
| 影响重复性的原因 | — | 金属网位置难固定；4 段注水比 2 段复杂 |

作者坦言 Stagg [X]"不是理想解决方案，但是目前家用环境下最好的选项之一"。在连续使用超过 400 杯后，他仍首选 Stagg [X]，因为**风味的提升远超重复性的损失**。
