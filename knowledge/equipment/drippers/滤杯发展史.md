# 咖啡滤杯发展史

## 滤杯诞生前：煮咖啡时代

- 咖啡从埃塞俄比亚传至也门 → 阿拉伯 → 土耳其奥斯曼帝国 → 意大利 → 欧洲北部
- 早期全程以**煮**的方式饮用：咖啡磨成细粉，放小壶里煮（土耳其咖啡延续至今）
- 传入欧洲后出现**渗滤壶**（类似越南滴滤壶），用两层筛网夹粉，靠重力缓慢滴滤，出品较浓郁
- 越往北欧洲口味越淡，推动了过滤方式的演变

## 关键节点一：美利塔（Melitta）—— 滤纸发明

**时间**：1908年

**人物**：德国人 **Melitta Bentz**（美利塔夫人）

**背景**：
- 美利塔出生于萨克森，当地人历来喝茶口味偏淡，对传统咖啡的苦涩和渣滓感到厌倦
- 某天从孩子的吸墨纸（pen blotter）获得灵感

**发明**：
- 在马克杯底部戳孔，剪一张圆形吸墨纸放入，倒上咖啡粉注水
- 得到一杯前所未有的**干净、清澈**的咖啡
- 1910年前后量产，在德国快速普及

**历史意义**：
- 世界上**第一个咖啡滤杯**
- 开创了纸滤过滤方式，清晰度从此成为手冲咖啡的核心价值
- 德国瓷器传统（梅森白瓷）使陶瓷滤杯成为标配，美利塔标志以蓝色印刷（受中国青花瓷影响）

**滤杯形态**：
- 早期：圆形垫片滤纸（配压纸器）
- 后来：单弧形滤纸，梯形（平底）滤杯
- 代表了梯形滤杯的起源

## 关键节点二：Chemex —— 锥形滤杯的起源

**时间**：1941年

**人物**：**Peter Schlumbohm**（德裔犹太化学博士，30年代纳粹排犹后移民美国）

**灵感来源**：美利塔，但 Schlumbohm 认为美利塔设计太过实用、缺乏美感

**设计理念**：
- 化学实验室玻璃器皿的造型语言
- 腰部收窄的瓶形（据说灵感来自女性曲线）
- **锥形滤杯** + 专用厚滤纸
- 一体式设计（上下合为一体，无单独滤杯/下壶之分）

**特点**：
- 锥形滤杯成为主流形态的**奠基者**
- 厚滤纸 + 无肋骨结构 → 流速极慢
- 入水注到水粉比即停，自然滴完，适合标准化出品（美国咖啡馆常用）
- 被纽约现代艺术博物馆（MoMA）永久收藏
- 在北美咖啡文化中具有标志性地位（星巴克"臻选"手冲长期使用Chemex）

**局限**：
- 无法控制流速，咖啡师无法通过手法调节风味
- 在精品咖啡圈逐渐被 V60 取代

**历史地位**：
- "如果没有 Chemex，可能就没有锥形滤杯，也就没有后来的 Hario"
- 美国精品咖啡手冲文化早期的核心器皿

## 关键节点三：Hario V60 —— 当代手冲的主流

**背景**：Chemex 带来的锥形形态在美国流行，但精品咖啡发展需要更可控的萃取工具

**Hario V60 的突破**：
- 60°锥角（V60 之名由此而来）
- 大单孔 + 螺旋肋骨 → 流速可被手法调控
- 咖啡师可以通过注水节奏、水柱粗细控制萃取
- 轻便、可量产，成为全球精品咖啡标配

## 手冲咖啡在日本的传承

- 手冲咖啡（Pour Over）在东亚特别盛行（日本、韩国、中国）
- 核心原因：东亚有喝茶的传统，口味偏淡，对干净清澈的饮品天然亲近
- **蓝瓶咖啡（Blue Bottle）** 创始人受日本大坊咖啡仪式感启发，将手冲文化带入美国第三波精品咖啡浪潮

## 精品咖啡器具演进时间线

| 年份 | 事件 |
|------|------|
| 1908 | 美利塔夫人发明咖啡滤纸和第一个滤杯（德国） |
| 1910年代 | 美利塔滤杯在德国普及 |
| 1941 | Chemex 发明，锥形滤杯形态确立（美国） |
| 2000 | WBC 世界咖啡师大赛首届，精品咖啡运动正式推动 |
| 2010后 | 精品咖啡爆发增长期，V60、Kalita 等手冲滤杯全球流行 |
| 2015 | 中国精品咖啡市场开始蓬勃发展 |

## 滤杯材质

- **陶瓷**：历史最悠久，比热容高，有热惰性，保温但预热慢；德国梅森白瓷最早实现量产白瓷滤杯
- **玻璃**：Chemex 使用，视觉美观
- **树脂（亚克力/PC）**：隔热效果好，温度流失少，但缺乏质感
- **金属（不锈钢/铜）**：导热快，温度传递迅速
