> 滤纸会同时影响流速、细粉和油脂截留、杯中干净度与风味呈现；不要只用“快/慢”判断一张滤纸。

## 测试的滤纸类型

| 类型 | 品牌 / 款型 |
|------|-----------|
| V60 纸质（带折） | Hario 漂白 / 未漂白 |
| V60 纸质（无折） | Hario 漂白 / 未漂白 |
| V60 纸质 | Cafec 漂白 |
| V60 布质 | Coffee Sock |
| 爱乐压 | 标准款 / Aesir 款 |
| Chemex | 漂白 / 未漂白 |
| 金属网 | Osaka |
| 虹吸壶 | 纸质 / 布质 |
| 实验室定量滤纸 | Whatman Grade 597 |

## 测量方法

- 图像分辨率：约 67.56 微米/像素（用校准网格标定）
- 孔隙分析：Butterworth 滤波 + 图像处理，计算孔径分布和均匀度
- 厚度测量：700+ 个测量点
- 流速测试：纯水（不加咖啡粉）实测

## 关键数据

### 纯水流速（ml/秒）

| 滤纸 | 流速 |
|------|------|
| Cafec V60 | 5.79（最慢纸质） |
| Hario 无折 | 6.89 |
| Hario 有折 | 11.03 |
| Osaka 金属网 | 67.8（最快） |

### 厚度

| 滤纸 | 厚度 |
|------|------|
| 爱乐压标准款 | 120 μm（最薄） |
| 布质过滤器 | 690 μm（最厚） |

## 颠覆直觉的发现

### 漂白 vs 未漂白

漂白工艺反而**恶化了孔隙分布质量**——未漂白滤纸在孔隙均匀性上表现更好，理论上更不容易产生局部通道。

但漂白滤纸完全不释放溶解固体（0 ppm），而未漂白款约释放 5 ppm 物质。不过：**150g 热水预润湿后，几乎所有可溶物质都被去除**，两者在味觉上没有实质差异。

### 有折 vs 无折（Hario V60）

无折 Tabless 被厂商宣传为更优，但在多项指标上，**有折款（Tabbed）性能更好**：更低的堵塞指数，实际冲煮中流速更均匀。

## 孔隙均匀度与堵塞指数

- **均匀度指数**：未漂白款更优，表明流速分布更均匀，通道效应更少
- **堵塞指数**：未漂白款也更低，说明抗细粉堵塞能力更强

## 实际冲煮中的风味影响

滤纸的纯水流速和实际咖啡风味不能简单画等号。一次使用 11 种滤纸、3 种咖啡、多人盲测的对比显示：流速相近的滤纸，杯中仍可能出现明显不同的酸质、甜感、沉闷感和苦涩感。

更稳妥的理解是：

- **流速只是结果之一**：滤纸材质、纤维结构、厚度、孔隙分布和与滤杯的贴合都会影响萃取。
- **不同咖啡适合不同滤纸**：高密度浅烘水洗豆可能能承受更长接触时间；重发酵或中烘豆则可能更需要降低沉闷、苦涩和厚重感。
- **滤纸可以和研磨联动**：高速滤纸不一定只能带来清淡杯感，也可以配合更细研磨，让粉水接触时间回到合适范围。
- **不要把单一测试结果当成品牌排名**：同一张滤纸在浸泡式、V60、平底滤杯或不同水质下可能表现不同。

因此，滤纸更像一个低成本调味变量：当研磨和水温已经接近合适区间，但杯中仍然闷、涩、薄或不够清晰时，可以尝试更换滤纸，而不是只继续大幅调整研磨。

## 纸滤、金属滤网与油脂

纸滤会截留大量油脂和细粉，因此杯中通常更干净、轻盈，沉淀更少。金属滤网会保留更多油脂和悬浮细粉，口感更厚，香气和触感也可能更强，但杯中浑浊感、沉淀和压舌感风险更高。

从脂质角度看，咖啡中的油脂主要来自生豆内胚乳，烘焙后仍会保留。未经纸滤过滤的冲煮方式，如法压壶、金属滤网、土耳其咖啡等，会让更多脂质进入杯中；纸滤则能显著降低这部分物质。选择纸滤或金属滤网时，关键不是“哪种更高级”，而是想要更干净轻盈，还是更厚重饱满。

## 最终推荐

| 应用场景 | 推荐 |
|---------|------|
| V60 手冲 | 漂白或未漂白都可用；未漂白款需充分预润湿，重点看实际杯感和流速稳定性 |
| 想提高干净度 | 优先纸滤，减少油脂和细粉进入杯中 |
| 想提高厚度 | 可尝试金属滤网或较高油脂保留的过滤方式，但要接受沉淀和浑浊风险 |
| 爱乐压 | 可尝试更致密或更均匀的滤纸，必要时用更稳定的下压力量补偿慢流速 |
| 调参思路 | 更换滤纸后重新确认研磨和总时间，不要沿用原参数直接判断好坏 |

## 局限性

- 测试使用干燥滤纸；润湿后孔隙分布可能有变化
- 均匀度指数测量的是毫米级别的变化，非微观不均匀性
- 实际冲煮结果受研磨粗细、手法等更多变量影响
