> Jonathan Gagné 用 USB 显微镜和图像处理算法对 13 款滤纸做了定量评测，结论颠覆直觉：漂白滤纸的孔隙质量不如未漂白款，充分预润湿可以消除纸味。

## 测试的滤纸类型

| 类型 | 品牌 / 款型 |
|------|-----------|
| V60 纸质（带折） | Hario 漂白 / 未漂白 |
| V60 纸质（无折） | Hario 漂白 / 未漂白 |
| V60 纸质 | Cafec 漂白 |
| V60 布质 | Coffee Sock |
| 爱乐压 | 标准款 / Aesir 款 |
| Chemex | 漂白 / 未漂白 |
| 金属网 | Osaka |
| 虹吸壶 | 纸质 / 布质 |
| 实验室定量滤纸 | Whatman Grade 597 |

## 测量方法

- 图像分辨率：约 67.56 微米/像素（用校准网格标定）
- 孔隙分析：Butterworth 滤波 + 图像处理，计算孔径分布和均匀度
- 厚度测量：700+ 个测量点
- 流速测试：纯水（不加咖啡粉）实测

## 关键数据

### 纯水流速（ml/秒）

| 滤纸 | 流速 |
|------|------|
| Cafec V60 | 5.79（最慢纸质） |
| Hario 无折 | 6.89 |
| Hario 有折 | 11.03 |
| Osaka 金属网 | 67.8（最快） |

### 厚度

| 滤纸 | 厚度 |
|------|------|
| 爱乐压标准款 | 120 μm（最薄） |
| 布质过滤器 | 690 μm（最厚） |

## 颠覆直觉的发现

### 漂白 vs 未漂白

漂白工艺反而**恶化了孔隙分布质量**——未漂白滤纸在孔隙均匀性上表现更好，理论上更不容易产生局部通道。

但漂白滤纸完全不释放溶解固体（0 ppm），而未漂白款约释放 5 ppm 物质。不过：**150g 热水预润湿后，几乎所有可溶物质都被去除**，两者在味觉上没有实质差异。

### 有折 vs 无折（Hario V60）

无折 Tabless 被厂商宣传为更优，但在多项指标上，**有折款（Tabbed）性能更好**：更低的堵塞指数，实际冲煮中流速更均匀。

## 孔隙均匀度与堵塞指数

- **均匀度指数**：未漂白款更优，表明流速分布更均匀，通道效应更少
- **堵塞指数**：未漂白款也更低，说明抗细粉堵塞能力更强

## 最终推荐

| 应用场景 | 推荐 |
|---------|------|
| V60 手冲 | **未漂白滤纸**（充分预润湿消除纸味，孔隙均匀性更好） |
| 金属网 | **不推荐**：孔径太大，迫使极细研磨，细粉穿透，杯中浑浊感重 |
| 布质过滤器 | **不推荐**：维护麻烦（每次需 OxiClean 煮沸），且风味影响不稳定 |
| 爱乐压 | **Aesir 滤纸** > 标准款（均匀度和堵塞指数更好，可通过更用力下压补偿慢流速） |

## 局限性

- 测试使用干燥滤纸；润湿后孔隙分布可能有变化
- 均匀度指数测量的是毫米级别的变化，非微观不均匀性
- 实际冲煮结果受研磨粗细、手法等更多变量影响
