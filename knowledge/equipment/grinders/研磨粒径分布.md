> 粒径分布（PSD）是评价磨豆机最客观的指标——峰值是否集中（单峰性）和粒径是否均匀，直接决定萃取均匀性。

## 什么是粒径分布（PSD）

研磨后的咖啡粉由不同粒径的颗粒组成，PSD 描述了各粒径区间的比例。理想磨豆机产出的 PSD 越集中（单峰、窄分布），萃取越均匀。

## 家用测量工具：Jonathan Gagné 开发的 App

**原理**：将咖啡粉均匀铺在白纸上，拍照后由软件进行图像分析：
- 识别独立颗粒（含相互接触颗粒的分离）
- 计算每颗粒子的投影面积（不假设球形，更准确）
- 生成粒径分布直方图和比较分析

**平台**：macOS / Windows / Linux（Python 3）；后有第三方制作简化安装包。

**关键限制**：相机分辨率决定了可检测的最小粒径（低于 1 像素的细粉无法检测）。

**初步发现**（Baratza Forté vs Lido 3）：Forté（54mm 平刀）产生的细粉（<0.5mm）远少于 Lido 3（48mm 锥刀），大颗粒也更少，说明平均粒径相近但粒径分布更窄。

## 24 款意式磨豆机大规模研究

Kaffeemacher 团队与 ZHAW 咖啡卓越中心使用 Camsize X2 成像设备，对 24 款意式磨豆机采集了 300 份粒径分布数据。

### 核心指标

| 指标 | 说明 |
|------|------|
| 细粉比例 | 小于 100 微米的颗粒占比 |
| 单峰性 | 分布曲线集中于单一峰值的程度 |
| 颗粒均匀度 | 峰值宽窄，越窄越均匀 |
| 粗粒尾部 | 大粒径的分布宽度 |

### 平刀 vs 锥刀的结论

行业内长期争论平刀/锥刀优劣，此研究给出了更细腻的答案：

> "锥刀平均而言单峰性更低、均匀度更差，但存在大量重叠——可以轻松找到某款平刀比某款锥刀更差。"

关键结论：**刀盘几何形状比刀盘尺寸对粒径均匀性影响更大。转速（RPM）与粒径均匀性无明显相关性。**

### 反直觉发现：单峰越好，需要更多细粉

单峰性越好的磨豆机在调试时，反而需要研磨到**略高的细粉含量**才能达到理想萃取。原因是：咖啡师在调试时主要调整细粉量（而非峰值粒径），单峰磨豆机的调试空间更窄。

### 味觉评分与 PSD 的关系

评分最高的磨豆机并非单峰性最极端的，而是在单峰性和均匀度之间取得平衡的款型。这说明 PSD 参数之外还有其他因素影响意式浓缩品质。

## 对冲煮实践的启示

| 问题 | 启示 |
|------|------|
| 如何选磨豆机 | 不能只看刀盘大小或类型，需参考实测 PSD 数据 |
| 调试意式浓缩 | 主要通过细粉量调整，而非单纯看研磨刻度数字 |
| 手冲 vs 意式 | 手冲对细粉更宽容，意式的高压使细粉的负面影响更突出 |
| 换磨豆机后需重新调试 | 不同磨豆机的细粉分布特征不同，参数不可直接迁移 |
