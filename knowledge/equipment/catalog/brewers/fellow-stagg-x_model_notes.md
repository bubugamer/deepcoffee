# Fellow Stagg X 模型注释

## 识别边界

- `Stagg X` 与 `Stagg XF` 是不同型号，不要自动合并。
- 用户只说 `Stagg` 时，如果没有更多上下文，保存为用户原文或追问型号。
- `Fellow` 是品牌，不是具体滤杯型号。

## 录入策略

- 明确出现 `X` 时保存 `Fellow Stagg X`；明确出现 `XF` 时保存 `Fellow Stagg XF`。
