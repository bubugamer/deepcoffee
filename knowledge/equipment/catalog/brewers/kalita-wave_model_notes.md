# Kalita Wave 模型注释

## 识别边界

- `Kalita Wave`、`Kalita`、`蛋糕杯` 可指同一类平底波浪滤杯。
- 用户没有说尺寸时保存通用 `Kalita Wave`，不要默认 `155` 或 `185`。
- 明确说 `155`、`185` 时，才保存对应尺寸项。

## 录入策略

- `Kalita Wave 滤纸` 是过滤介质或耗材，不要误存为滤杯。
