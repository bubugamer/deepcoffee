> 天体物理学家，同时是全球最具影响力的咖啡科学研究者之一，以定量方法研究手冲与意式浓缩的物理机制。

## 基本信息

| 项目 | 内容 |
|------|------|
| 职业 | 天体物理学家（研究方向：年轻恒星、褐矮星、系外行星） |
| 咖啡博客 | [Coffee ad Astra](https://coffeeadastra.com) |
| 博客含义 | "ad Astra"为拉丁语"直达星空"，同时呼应天文背景 |
| 著作 | *The Physics of Filter Coffee*（手冲）；*The Physics of Espresso*（2025 年出版） |

## 在咖啡领域的贡献

### 科学量化方法
将天体物理学中的统计学、实验设计和量化分析引入精品咖啡领域：
- 开发了咖啡研磨粒径分布测量软件（免费开源）
- 建立了手冲萃取率的精确测量方法（玻璃吸管 + 温控）
- 用统计显著性框架评估盲测结果

### 器具设计
与 NextLevel Brewer 联合设计了 **Pulsar 滤杯**和 **Pulsar Mini**，集成阀门控流、圆柱无绕流、分散网等设计，获销售版税。

### 核心研究主题

| 主题 | 代表成果 |
|------|---------|
| 萃取均匀性 | 粉床下纸质过滤、WDT、Weber Shaker 布粉方式 |
| 涩感机制 | 提出涩感是过滤问题而非萃取问题的理论框架 |
| 粉床深度 | 系统梳理粉床深度对流速、过滤、萃取均匀性的四重影响 |
| 用水科学 | 整理 SCA 参数 + DIY 配水方案 + 配水计算工具 |
| 粒径分布 | 开发开源测量软件；参与 24 款意式磨豆机研究分析 |
| 风味数据 | 用 1500 个样本分析品种与处理法的风味图谱 |

## 写作风格

Gagné 的文章以**数据优先、承认局限性**著称：
- 每篇文章均包含实验设计说明和方法论局限
- 愿意报告反直觉的发现（如未漂白滤纸孔隙质量更好，极细研磨反而不漏粉）
- 定量数据与感官体验并重
